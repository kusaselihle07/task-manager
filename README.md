# Personal Task Manager (CRUD App)

A simple, real-world web app to **Add, Read, Update, and Delete (CRUD) tasks**, built with **HTML, CSS, and JavaScript**. Tasks are stored in **localStorage** so your data persists even after refreshing the page.

---

## **ðŸ”¹ Features**

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as **complete** or undo completion
- Sorts tasks: active tasks first, completed tasks last
- Shows **Total Tasks** and **Completed Tasks**
- Persistent storage using **localStorage**
- Fully responsive and professional design
